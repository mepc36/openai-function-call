# Summary

This repo is a minimal example. It shows how to call an openAI function call.

## Running the App:

1. Install dependencies:

```
yarn install
```

2. Set up your .env file:

```
cp .env.example .env
```

3. Add your OPENAI_API_KEY value to your .env file.

4. Run the script:

```
node call-openai-function.js
```
